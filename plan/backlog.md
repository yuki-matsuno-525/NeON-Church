## 設計方針（全ステップ共通）

| 項目 | 方針 |
|------|------|
| ページネーション方式 | ページ番号型（URL: `?page=N`）。全リスト系で統一 |
| 未ログイン時の誘導 UI | モーダル表示。ページ遷移なし、元の操作に戻れる |
| スマホコメントパネルの URL | 変更あり（例: `?panel=comments`）。ブラウザバックで閉じられる |
| 通知バッジの非同期更新 | WebSocket または SSE |
