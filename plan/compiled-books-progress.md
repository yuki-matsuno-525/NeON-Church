# 編纂書機能 実装進捗

## ゴール

NeON Church上の既存テキストから節を集め、普通の本文も追加し、章にまとめ、ひとつの「編纂書」として下書き保存・公開できる機能を完成させる。

## MVP仕様

- [x] 編纂書を作成できる（Backend API）
- [x] 作業途中の下書きが保存される（Backend API）
- [x] 書に説明と注釈を付けられる（Backend API）
- [x] 章を作成できる（Backend API）
- [x] 章に導入文と注釈を付けられる（Backend API）
- [x] 既存本文の節を未整理トレイへ追加できる（Backend API）
- [x] 普通の本文を編纂書内の節として追加できる（Backend API）
- [x] 未整理トレイの節を章へ移動できる（Backend API）
- [x] 節に注釈を付けられる（Backend API）
- [x] 章内の節番号が自動的に振られる（Backend API）
- [x] private / unlisted / public の公開範囲を扱える（Backend API）
- [x] 公開された編纂書を読める
- [x] 編纂書の書・章・節にコメントできる（Backend API）
